_ Other DB _

* PostgreSQL: 
	+ Install: 
		- $ sudo apt install -y postgresql 
	+ Service: 
		- $ sudo systemctl start postgresql 
	+ Port: 5432. 
	+ Confirm: 
		- $ psql --version
