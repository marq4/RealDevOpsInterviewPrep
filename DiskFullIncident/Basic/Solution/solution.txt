	6.- | 
		+ Overlay is Docker's default storage driver (filesystem layer). 
		+ It's how Docker efficiently stores container filesystems. 
		+ Container fs (overlay): 
			- TOP: container layer (RW). 
			- Image layers (R). 
			- Host fs. 
		+ Overlay combines all layers into a single unified fs that the container sees.
